# 国立・都立小学校受験 単元一覧

対象校: 筑波、お茶の水、学芸大附属各校（大泉・小金井・世田谷・竹早）、立川国際

## 1. ペーパーテスト

### 図形領域
| 単元 | 内容 | 対象校 |
|------|------|--------|
| 回転図形 | お手本の図形が回転した後の正しい形を選ぶ | 筑波など |
| 重ね図形（折り重ね図形） | 複数の図形を重ねた時、折って重ねた時の見え方を推理 | 筑波など |
| 図形構成（パズル） | ピースを組み合わせてどの形になるか、お手本を分けたピースを描き足す | 筑波、小金井など |
| 位置の移動 | 矢印の指示に従い最終的な位置を答える | 筑波 |
| 四方観察 | 立体を別の角度から見た時の見え方を選ぶ | 筑波、お茶の水 |
| 模写（点図形・フリーハンド） | お手本と同じ図形や点の繋がりを書き写す | 世田谷など |
| 同図形・異図形発見 | 細かい違いを見分ける、同じものを見つける | - |
| 鏡図形 | 鏡に映した時の図形を推理 | - |
| 系列完成 | パターンの規則性を見つけて続きを答える | - |
| 折り目の推理 | 折った紙を切った時の展開図を推理 | - |
| 立体図形 | 立体の構成や展開図に関する問題 | - |
